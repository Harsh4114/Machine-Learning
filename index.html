Types of Machine Learning

1. Supervised Learning

You can train an algorithm given input 'x' to predict output 'y'.

Example: Spam mail and automatic car

2. Unsupervised Learning

You're dealing with data that isn't associated with any labels.

3. Reinforcement Learning

An agent learns to make decisions by taking actions and evaluating the outcomes through rewards and penalties.
